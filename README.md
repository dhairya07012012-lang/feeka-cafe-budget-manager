# 🍽️ Feeka Cafe - Budget Manager

Complete financial management system with AI-powered insights, dark/light theme, and persistent data storage.

## ✨ Features

### 🎨 Theme Toggle
- Switch between Dark and Light mode
- Comfortable viewing in any lighting

### 🕐 Live Date & Time
- Current date and time displayed
- Top right corner, always visible

### 💾 Data Persistence
- All data saves automatically
- Export/Import JSON backups
- Data survives page refreshes

### 📊 Financial Management
- Revenue tracking by source
- Expense tracking by category
- Visual charts and graphs
- Profit & Loss statements

### 🤖 AI Finance Advisor
- Powered by GPT-4
- Analyzes YOUR actual data
- Personalized recommendations
- Consultation history
